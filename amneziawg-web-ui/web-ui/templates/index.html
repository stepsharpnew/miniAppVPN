⚠ Об обходе белых списков (Орёл, МТС, Мегафон)

ТСПУ в режиме белых списков замораживает все TCP-сессии к зарубежным IP после 15–20 KB — не RST, а именно freeze, соединение виснет. VLESS+Reality на зарубежном VPS в этом режиме не работает вне зависимости от протокола.

Решение — цепочка (chain): клиент → российский VPS с «белым» IP (Yandex.Cloud / EDGE CDN) → зарубежный VPS → интернет. Российский узел настраивается как relay через vnext в Xray outbound. Трафик внутри страны ТСПУ не замораживает.

Проверить, «белый» ли IP у хостинга: запустите на порту 443 веб-сервер и откройте сайт с мобильного МТС/Мегафон без VPN.

Рекомендуется. nginx stream проксирует внешний :443 к Xray по SNI. Нужен SSL (SSL_DOMAIN). Обходит ISP, которые фильтруют по SNI-имени, но не по IP. Без галочки — Xray слушает на 8443+.

Должен резолвиться на этот сервер; используется в vless:// и HTTP Host.

packet-up: каждый чанк — отдельный HTTP POST (как загрузка файла в браузере). Труднее фингерпринтить DPI.

Какой браузер имитирует TLS-хэндшейк. ios/safari — для Apple-устройств.

host:port реального TLS-сайта (REALITY будет маскироваться под него). Для белых списков — выбирайте сайты из белого списка вашего ISP (госуслуги, сбербанк, vk.com и т.п.).

Генерация relay-конфига для обхода белых списков

Схема подключения:

Клиент → Российский VPS (белый IP) → Зарубежный VPS (этот сервер) → Интернет

Только IP-адрес (домен не нужен). Должен быть «белым».
